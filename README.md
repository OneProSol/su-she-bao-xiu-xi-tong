# 宿舍报修系统

一个基于Vue2 + SpringBoot + MyBatisPlus + MySQL的宿舍报修管理系统。

### 一、相关技术
- 后端：`jdk`、`SpringBoot 2.x`、`MyBatisPlus 3.x`、`JWT`。
- 前端：`node`、`Vue 2.6`、`Vue Router`、`Element UI`、`Axios`。
- 数据库：`mysql`、`Redis`。

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk17 或 jdk1.8`。
- 前端：`node`、`vue-cli` 、`npm` 。
- 数据库：`mysql5.7以上`、`Redis`。

### 四、功能描述及功能图展示

系统分为：学生、维修人员、管理员。

学生功能：1.登录注册、2.发起报修、3.我的报修、4.楼栋查看、5.意见反馈、6.个人信息

维修人员功能：1.登录、2.查看公告、3.维修任务、4.我的评价、5.个人信息

管理员功能：1.登录、2.数据统计、3.公告管理、4.学生管理、5.维修人员管理、6.注册审核、7.报修单管理、8.报修类型管理、9.楼栋管理、10.房间管理、11.宿舍变更审核、12.意见反馈管理
